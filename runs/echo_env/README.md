# Echo Env

A simple run that demonstrates how to access environment variables. Returns the value of the `test` environment variable.

## Environment Variables

| Name | Type | Description |
|------|------|-------------|
| `test` | text | The environment variable value to echo back |

## Inputs

This run has no inputs.

## Response

Returns the value of the `test` environment variable.
